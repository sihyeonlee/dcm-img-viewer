# DCM-IMG-Viewer

## About
![DCM-IMG-Viewer](/img/program_image.png)
- Basic handling of Dicom Files, using pydicom library.
- Extract Image from .dcm file and modulate with Window Center(WC) and Window Width(WW).

## Environment
- python3 ( 3.7.9 )
    - pydicom
    - pyside2
    - numpy
    - matplotlib
